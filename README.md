# Glow

Proyecto web con panel cliente, negocio y admin.

## Local

1. Crear `.env` con `DATABASE_URL`.
2. Instalar dependencias con `npm install`.
3. Ejecutar `npm start`.
4. Abrir `http://localhost:3001`.

## Vercel

1. Importar el repositorio en Vercel.
2. Configurar la variable de entorno `DATABASE_URL`.
3. Desplegar.

Vercel servira los archivos de `public/` como frontend y `api/data.js` como endpoint para persistencia.
